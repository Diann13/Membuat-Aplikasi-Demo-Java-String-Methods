## 📖 Tentang Project

**Java String Methods Demo** merupakan aplikasi berbasis **Java Console** yang dibuat sebagai media pembelajaran mengenai berbagai method pada class `String` di Java.

Project ini saya kerjakan ketika masih menempuh pendidikan di **SMK Negeri 1 Pacitan** jurusan **Rekayasa Perangkat Lunak (RPL)** sebagai salah satu tugas praktik pemrograman Java.

Repository ini saya simpan sebagai dokumentasi perjalanan belajar saya di dunia pemrograman sekaligus sebagai bagian dari portofolio perkembangan kemampuan saya.

# 📚 Penjelasan Method

| Method | Fungsi |
|---------|--------|
| `charAt()` | Mengambil karakter berdasarkan posisi index |
| `concat()` | Menggabungkan dua buah String |
| `substring()` | Mengambil sebagian isi String |
| `toUpperCase()` | Mengubah seluruh huruf menjadi kapital |
| `toLowerCase()` | Mengubah seluruh huruf menjadi huruf kecil |
| `trim()` | Menghapus spasi di awal dan akhir String |
| `indexOf()` | Menampilkan posisi suatu substring |
| `isEmpty()` | Mengecek apakah String kosong |
